#### Requirements

        To install Sonatype Nexus, your server must meet the following requirements:

        - Windows Server 2022 or later.
        - 4 CPU cores
        - 16 GB RAM
        - 500 GB of drive space for package storage. We suggest using a separate drive so the storage can be expanded if necessary.
